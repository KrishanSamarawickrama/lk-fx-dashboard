# Contributor Covenant Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

Instances of unacceptable behavior may be reported by opening a
[GitHub Issue](https://github.com/krishans1990/lk-fx-dashboard/issues)
or contacting the project maintainers directly.

All complaints will be reviewed and investigated promptly and fairly.
